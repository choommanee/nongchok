## การเข้าสู่ระบบ

### วิธีการเข้าสู่ระบบ:
1. เปิดเบราว์เซอร์ไปที่: `https://yourdomain.com/wp-admin`
2. กรอก Username และ Password
3. คลิก "เข้าสู่ระบบ"

### หน้า Dashboard:
- แสดงภาพรวมของเว็บไซต์
- แจ้งเตือนการอัพเดต
- สถิติการเข้าชมเว็บ

## การจัดการหน้าแรก (Home)

### การแก้ไขเนื้อหาหน้าแรก:
1. ไปที่เมนู **Pages > All Pages**
2. ค้นหา "Home" หรือ "หน้าแรก"
3. คลิก **Edit**
4. แก้ไขเนื้อหาในส่วนต่างๆ:
   - Hero Section (ภาพและข้อความหลัก)
   - Welcome Message
   - Featured Services
   - Gallery Preview

### การเปลี่ยนรูปภาพ Hero:
1. คลิกที่รูปภาพในตัวแก้ไข
2. เลือก **Replace Image**
3. อัพโหลดรูปใหม่หรือเลือกจาก Media Library
4. แนะนำขนาด: 1920x1080px

---

## การจัดการหน้าเกี่ยวกับเรา (About Us)

### การแก้ไขเนื้อหา:
1. ไปที่ **Pages > All Pages**
2. ค้นหา "About Us" หรือ "เกี่ยวกับเรา"
3. คลิก **Edit**
4. แก้ไขส่วนต่างๆ:
   - ประวัติบริษัท
   - วิสัยทัศน์และพันธกิจ
   - ทีมงาน

### การเพิ่มรูปทีมงาน:
1. คลิก **Add Media**
2. อัพโหลดรูปทีมงาน
3. แนะนำขนาด: 400x400px (รูปสี่เหลี่ยมจัตุรัส)
4. ใส่ชื่อและตำแหน่งในช่อง Caption

## การจัดการบริการ (Service)

### การเพิ่มบริการใหม่:
1. ไปที่เมนู **Services** (หรือ **บริการ**)
2. คลิก **Add New**
3. กรอกข้อมูล:
   - **ชื่อบริการ**: ชื่อบริการที่ต้องการแสดง
   - **รายละเอียด**: คำอธิบายบริการ
   - **Featured Image**: รูปภาพประจำบริการ (600x400px)
   - **ราคา**: ใส่ราคาบริการ (ถ้ามี)
4. คลิก **Publish**

### การแก้ไขบริการ:
1. ไปที่ **Services > All Services**
2. เลือกบริการที่ต้องการแก้ไข
3. คลิก **Edit**
4. แก้ไขข้อมูลตามต้องการ
5. คลิก **Update**

### การลบบริการ:
1. ไปที่ **Services > All Services**
2. วางเมาส์เหนือบริการที่ต้องการลบ
3. คลิก **Trash**

---

## การจัดการข่าวสาร (News)

### การเพิ่มข่าวใหม่:
1. ไปที่เมนู **Posts > Add New** หรือ **News > Add New**
2. กรอกข้อมูล:
   - **หัวข้อข่าว**: ชื่อข่าวที่ต้องการ
   - **เนื้อหา**: รายละเอียดข่าว
   - **หมวดหมู่**: เลือกหมวดหมู่ข่าว
   - **Featured Image**: รูปภาพประกอบข่าว (1200x628px)
   - **Tags**: คีย์เวิร์ดที่เกี่ยวข้อง
3. กำหนด **วันที่เผยแพร่**:
   - Publish immediately: เผยแพร่ทันที
   - Schedule: ตั้งเวลาเผยแพร่ล่วงหน้า
4. คลิก **Publish** หรือ **Schedule**

### การแก้ไขข่าว:
1. ไปที่ **Posts > All Posts** หรือ **News > All News**
2. ค้นหาข่าวที่ต้องการ
3. คลิก **Edit**
4. แก้ไขเนื้อหา
5. คลิก **Update**

### การจัดการหมวดหมู่ข่าว:
1. ไปที่ **Posts > Categories**
2. **เพิ่มหมวดหมู่ใหม่**:
   - ใส่ชื่อหมวดหมู่
   - ใส่ Slug (URL)
   - เลือก Parent Category (ถ้ามี)
   - คลิก **Add New Category**

## การจัดการแกลเลอรี่ (Gallery)

### ระบบแกลเลอรี่มี 3 ประเภท:
1. **Gallery** - รูปภาพทั่วไป
2. **Ayam List** - รายการไก่ชน
3. **Behind the Scene (BTS)** - ภาพเบื้องหลัง

### การเพิ่มหมวดหมู่ใหม่:
1. ไปที่ **Gallery > Categories**
2. คลิก **Add New Category**
3. กรอกข้อมูล:
   - **Category Name**: ชื่อหมวดหมู่
   - **Category Number**: หมายเลขหมวดหมู่ (เรียงลำดับ)
   - **Category Type**: เลือกประเภท (gallery/ayam_list/behind_scene)
   - **Thumbnail**: อัพโหลดรูปปก
4. คลิก **Save Category**

### การเพิ่มรูปภาพในหมวดหมู่:
1. ไปที่ **Gallery > All Categories**
2. คลิก **Manage Images** ที่หมวดหมู่ที่ต้องการ
3. คลิก **Upload Images**
4. เลือกรูปภาพ (สามารถเลือกหลายรูปพร้อมกัน)
   - แนะนำขนาด: 1200x800px หรือมากกว่า
   - รองรับไฟล์: JPG, PNG, WebP
5. รูปภาพจะถูกอัพโหลดและแสดงในหมวดหมู่

### การจัดเรียงรูปภาพ:
1. ไปที่หน้าจัดการรูปภาพของหมวดหมู่
2. ลากและวางรูปภาพเพื่อเรียงลำดับใหม่
3. คลิก **Save Order**

### การลบรูปภาพ:
1. ไปที่หน้าจัดการรูปภาพ
2. คลิกปุ่ม **Delete** ที่รูปภาพที่ต้องการลบ
3. ยืนยันการลบ

### การจัดการรูปภาพแบบ Bulk:
1. เลือกรูปหลายๆ รูปโดยติ๊กช่อง checkbox
2. เลือกการกระทำ:
   - **Delete Selected**: ลบรูปที่เลือก
   - **Move to Category**: ย้ายไปหมวดหมู่อื่น
3. คลิก **Apply**

## การจัดการหน้าติดต่อ (Contact)

### การแก้ไขข้อมูลติดต่อ:
1. ไปที่ **Pages > All Pages**
2. ค้นหา "Contact" หรือ "ติดต่อเรา"
3. คลิก **Edit**
4. แก้ไขข้อมูล:
   - **ที่อยู่**: ที่อยู่บริษัท
   - **เบอร์โทร**: เบอร์ติดต่อ
   - **อีเมล**: อีเมลติดต่อ
   - **เวลาทำการ**: วัน-เวลาทำการ
   - **แผนที่**: Embed Google Maps
5. คลิก **Update**

### การเพิ่ม Google Maps:
1. ไปที่ Google Maps
2. ค้นหาสถานที่ของคุณ
3. คลิก **Share > Embed a map**
4. คัดลอกโค้ด iframe
5. วางในส่วน HTML ของหน้า Contact

### การจัดการ Contact Form:
1. ไปที่ **Contact > Contact Forms** (ถ้าใช้ Contact Form 7)
2. เลือกฟอร์มที่ต้องการแก้ไข
3. แก้ไขฟิลด์ต่างๆ:
   - ชื่อ-นามสกุล
   - อีเมล
   - เบอร์โทร
   - ข้อความ
4. ตั้งค่าอีเมลที่จะรับข้อความ
5. คลิก **Save**

## การจัดการเมนู

### การแก้ไขเมนูหลัก:
1. ไปที่ **Appearance > Menus**
2. เลือก **Main Menu** หรือ **เมนูหลัก**
3. **เพิ่มรายการเมนู**:
   - เลือกจาก Pages, Posts, หรือ Custom Links
   - คลิก **Add to Menu**
4. **จัดเรียงเมนู**:
   - ลากและวางเพื่อเรียงลำดับ
   - ลากไปทางขวาเพื่อทำเป็น Submenu
5. **แก้ไขชื่อเมนู**:
   - คลิกลูกศรที่รายการเมนู
   - แก้ไข Navigation Label
6. คลิก **Save Menu**

### การสร้าง Dropdown Menu:
1. เพิ่มรายการเมนูหลัก
2. เพิ่มรายการเมนูย่อย
3. ลากรายการย่อยไปทางขวาใต้เมนูหลัก
4. Save Menu

## การจัดการภาษา

### ระบบแปลภาษาอัตโนมัติ:
เว็บไซต์มีระบบแปลภาษาอัตโนมัติ 3 ภาษา:
- **ไทย (TH)** - ภาษาหลัก
- **อังกฤษ (EN)**
- **อินโดนีเซีย (ID)**

### การใช้งาน:
1. ผู้เข้าชมสามารถเปลี่ยนภาษาได้จากปุ่มด้านบนขวา (Desktop)
2. บน Mobile จะอยู่ในเมนู Hamburger ด้านล่าง
3. ระบบจะจำภาษาที่เลือกไว้

### การเพิ่มเนื้อหาหลายภาษา:
1. เขียนเนื้อหาเป็นภาษาไทยเป็นหลัก
2. ระบบ Google Translate จะแปลอัตโนมัติ
3. สำหรับเนื้อหาสำคัญที่ต้องการแปลเอง:
   - ใช้ Plugin เช่น Polylang หรือ WPML
   - สร้างเนื้อหาแยกสำหรับแต่ละภาษา

## เคล็ดลับและข้อควรระวัง

### การอัพโหลดรูปภาพ:
✅ **ควรทำ**:
- ลดขนาดรูปก่อนอัพโหลด (ใช้ TinyPNG.com)
- ตั้งชื่อไฟล์เป็นภาษาอังกฤษ
- ใช้รูปแบบ JPG สำหรับภาพถ่าย, PNG สำหรับโลโก้
- ขนาดไฟล์ไม่เกิน 2MB

❌ **ไม่ควรทำ**:
- อัพโหลดรูปขนาดใหญ่เกินไป (>5MB)
- ใช้ชื่อไฟล์ภาษาไทยหรือมีช่องว่าง
- อัพโหลดรูปที่มีลิขสิทธิ์

### การสำรองข้อมูล:
1. สำรองข้อมูลทุกสัปดาห์
2. ก่อนทำการเปลี่ยนแปลงใหญ่ๆ
3. ใช้ Plugin เช่น UpdraftPlus

### SEO:
1. ใส่ Meta Description ในทุกหน้า
2. ใช้ Alt Text สำหรับทุกรูปภาพ
3. URL ควรสั้นและมีความหมาย
4. ใช้ Heading Tags (H1, H2, H3) อย่างถูกต้อง

### ความปลอดภัย:
1. เปลี่ยนรหัสผ่านทุก 3 เดือน
2. ใช้รหัสผ่านที่ซับซ้อน
3. ไม่แชร์ข้อมูลเข้าสู่ระบบ
4. ออกจากระบบเมื่อใช้งานเสร็จ

## การแก้ไขปัญหาเบื้องต้น

### ⚠️ ปัญหา URL หลังบ้านผิด (wp-admin ไม่ทำงาน):
**อาการ**: ลิงค์ในหน้า admin ไม่มี /wp-admin/ หรือไปหน้าอื่น เช่น edit.php แทนที่จะเป็น /wp-admin/edit.php

**วิธีแก้ไข 3 วิธี**:

#### วิธีที่ 1: ใช้ Script แก้ไข (แนะนำ)
1. อัพโหลดไฟล์ `fix-wordpress-url.php` ไปที่ root ของเว็บ
2. เข้า: `https://nongchokayambangkok.com/fix-wordpress-url.php`
3. Login เป็น admin
4. ใส่ URL ที่ถูกต้อง:
   - Production: `https://nongchokayambangkok.com`
   - Staging: `https://nongchok-production.up.railway.app`
   - Local: `https://nongchok.local`
5. คลิก Update
6. **ลบไฟล์ทันทีหลังใช้งาน**

#### วิธีที่ 2: แก้ไขใน wp-config.php
เพิ่มบรรทัดนี้ใน `wp-config.php` (ก่อน "That's all, stop editing!"):
```php
// สำหรับ Production
define('WP_HOME', 'https://nongchokayambangkok.com');
define('WP_SITEURL', 'https://nongchokayambangkok.com');
```

#### วิธีที่ 3: แก้ผ่าน Database (phpMyAdmin)
1. เข้า phpMyAdmin
2. เลือก database ของเว็บ
3. ไปที่ตาราง `wp_options`
4. แก้ไข 2 แถว:
   - `siteurl` = `https://nongchokayambangkok.com`
   - `home` = `https://nongchokayambangkok.com`
5. Save

### เว็บไซต์โหลดช้า:
- ตรวจสอบขนาดรูปภาพ
- ล้าง Cache
- ติดต่อผู้ดูแลระบบ

### ไม่สามารถอัพโหลดรูปภาพ:
- ตรวจสอบขนาดไฟล์ (ไม่เกิน 10MB)
- ตรวจสอบนามสกุลไฟล์
- ตรวจสอบพื้นที่จัดเก็บ

### หน้าเว็บแสดงผิดปกติ:
- ล้าง Browser Cache
- ลองเปิดด้วย Browser อื่น
- ตรวจสอบการแก้ไขล่าสุด

### ไม่สามารถ Login เข้า Admin:
- ตรวจสอบ URL ว่าถูกต้อง: `/wp-admin`
- ลอง Reset Password
- ตรวจสอบ Cookie Settings
- ลบ Cache และ Cookie ของ Browser

## การอัพเดตระบบ

### การอัพเดต WordPress:
1. สำรองข้อมูลก่อนอัพเดต
2. ไปที่ **Dashboard > Updates**
3. คลิก **Update Now**
4. รอจนอัพเดตเสร็จ
5. ตรวจสอบการทำงานของเว็บ

### การอัพเดต Plugins:
1. ไปที่ **Plugins > Installed Plugins**
2. ดูรายการที่มีอัพเดต
3. คลิก **Update Now**
4. ตรวจสอบการทำงาน

### การอัพเดต Theme:
1. ไปที่ **Appearance > Themes**
2. ดู Theme ที่มีอัพเดต
3. คลิก **Update Now**
4. ตรวจสอบการแสดงผล

## ภาคผนวก

### ขนาดรูปภาพแนะนำ:
- **Hero Banner**: 1920x1080px
- **Gallery Images**: 1200x800px
- **Service Images**: 600x400px
- **Team Photos**: 400x400px
- **News Featured**: 1200x628px
- **Thumbnails**: 300x200px

### Plugin ที่จำเป็น:
1. **Yoast SEO** - จัดการ SEO
2. **Contact Form 7** - ฟอร์มติดต่อ
3. **UpdraftPlus** - สำรองข้อมูล
4. **Wordfence** - ความปลอดภัย
5. **WP Super Cache** - เพิ่มความเร็ว

### Shortcodes ที่ใช้บ่อย:
```
[gallery] - แสดงแกลเลอรี่
[contact-form] - แสดงฟอร์มติดต่อ
[services] - แสดงรายการบริการ
[latest-news count="3"] - แสดงข่าวล่าสุด 3 ข่าว
```
